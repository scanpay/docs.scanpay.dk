WooCommerce

We have developed a payment plugin for WooCommerce. WooCommerce is an open-source e-commerce plugin for WordPress. Our WooCommerce plugin is developed and maintained by our development team in Copenhagen. The source code is available on GitHub. The plugin is compatible with WooCommerce Subscriptions.

Installation

You can install the plugin directly from your WordPress. Log in to your WordPress admin area and navigate to Plugins > Add New. Search for "scanpay" and install the following plugin:

Configuration

  1. Navigate to Plugins > Installed Plugins and activate the "Scanpay for WooCommerce" plugin.
  2. Navigate to WooCommerce > Settings > Payments. This is where you manage your payment settings in WooCommerce. Find the payment method called "Scanpay" and click "Set up". This will open the configuration page, as shown on the pictures below.
  3. Insert your Scanpay API key in the "API key" field and click "Save changes". You can generate an API key in our dashboard (here). It is important to always keep your API key private and secure.
  4. Click the "send ping" button at the top of the configuration page. This will take you to our dashboard, where it will set your ping URL and send a manual ping to your shop.
  5. You have now completed the installation and configuration of our WooCommerce plugin. We recommend that you perform a test order, to ensure that everything is working as intended.

Frequently asked questions

  1. How to enable MobilePay Online?
    First you need to enable MobilePay Online in our dashboard (link). Then open WooCommerce and navigate to WooCommerce > Settings > Payments and enable "MobilePay (Scanpay)".
  2. Does this plugin support WooCommerce Subscriptions?
    Yes. The plugin is compatible with WooCommerce Subscriptions.
