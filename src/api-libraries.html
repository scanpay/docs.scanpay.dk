API Libraries

Scanpay has official API client libraries for different programming languages. Client libraries make it easier for you to integrate our APIs in your project. All client libraries follow the same scheme and provide the following three methods:

  • newURL: Create a new payment link (API)
  • seq: Make a sequence request (API)
  • handlePing: Verify and parse ping requests (API)

PHP

The PHP client library is available through Composer (packagist) and the source code is hosted at GitHub.

composer require scanpay/scanpay

Node.js

The node.js client library is available through npm and the source code is hosted at GitHub.

npm install scanpay

Ruby

The Ruby client library is available as a gem and the source code is hosted at GitHub.

gem install scanpay

Go

The Go client library is hosted at GitHub.

go get github.com/scanpaydk/go-scanpay
