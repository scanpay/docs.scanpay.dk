Security at Scanpay

We have built our system from the ground up, with a security by design approach, making security considerations the core of our engineering process. You can therefore rest assured that we have implemented multiple layers of security to protect your data. If you have any questions or concerns, please do not hesitate to contact us.

Data protection

We are fully compliant with the General Data Protection Regulation (GDPR) and the Payment Card Industry Data Security Standard (PCI DSS). Sensitive data is encrypted with AES-256-GCM and stored in multiple data centers within the European Union. The encryption key is protected with Shamir's Secret Sharing and only a few trusted employees have a share of the key. Three shares (employees) are needed to decrypt and access your data.

In our daily operations we share sensitive data with the following third parties:

Amazon Web Services, Inc. Amazon Web Services (AWS) is our primary hosting provider. They are PCI DSS certified and GDPR compliant. All your data is encrypted and stored in three AWS regions within the EU.

Transport Layer Security

We only support Transport Layer Security (TLS) version 1.2 and 1.3. All our domains are on the HSTS preload list. We use Elliptic Curve Cryptography and we support the following cipher suites:

Cipher TLS Key size Kx Mac
ECDHE-ECDSA-AES128-GCM-SHA256 v1.2 128 Bit ECDH AEAD
ECDHE-ECDSA-AES256-GCM-SHA384 v1.2 256 Bit ECDH AEAD
ECDHE-ECDSA-AES128-SHA256 v1.2 128 Bit ECDH SHA256
ECDHE-ECDSA-AES256-SHA384 v1.2 256 Bit ECDH SHA384

Basic authentication

All API requests have to be authenticated via HTTP Basic Auth with API keys as credentials. API keys are unique and they contain your shopid and a random key, i.e. shopid:key. If you use our API client libraries or e-commerce modules then we will take care of the authentication. "Authorization: Basic " + base64_encode(apikey)

PCI DSS

The Payment Card Industry Data Security Standard (PCI DSS) is a security standard for businesses that store, process or transmit credit cards. PCI DSS is a continuous process and since 2017, we have successfully completed two level 1 certifications and one SAQ D certification. You can view our current Attestation of Compliance (AOC) here.
