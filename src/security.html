Security at Scanpay

We have built our system from the ground up, with a security by design approach, making security considerations the core of our engineering process. You can therefore rest assured that we have implemented multiple layers of security to protect your data. If you have any questions or concerns, please do not hesitate to contact us.

Data protection

We are fully compliant with the General Data Protection Regulation (GDPR) and the Payment Card Industry Data Security Standard (PCI DSS). Sensitive data is encrypted with AES-256-GCM and stored in multiple data centers within the European Union. The encryption key is protected with Shamir's Secret Sharing and only a few trusted employees have a share of the key. This ensures that no single employee can access your data on their own.

We process and store sensitive information in the following locations:

Transmission security

For security reasons, we only support Transport Layer Security (TLS) version 1.2 and higher. All our websites are served with HTTP Strict Transport Security (HSTS) and we are on the Google HSTS preload list. We only support Elliptic Curve Cryptography and the following cipher suites:

Cipher TLS Kx Mac
AES_256_GCM_SHA384 v1.3 ECDH AEAD
CHACHA20_POLY1305_SHA256 v1.3 ECDH AEAD
AES_128_GCM_SHA256 v1.3 ECDH AEAD
ECDHE_ECDSA_WITH_AES_128_GCM_SHA256 v1.2 ECDH AEAD
ECDHE_ECDSA_WITH_AES_256_GCM_SHA384 v1.2 ECDH AEAD
ECDHE-ECDSA-AES128-SHA256 v1.2 ECDH SHA256
ECDHE-ECDSA-AES256-SHA384 v1.2 ECDH SHA384

Internal data transmission is encrypted with AES-256 and tunneled through wireguard.

API authentication

All API requests have to be authenticated via HTTP Basic Auth with API keys as credentials. API keys are unique and they contain your shopid and a random key, i.e. shopid:key. If you use our API client libraries or e-commerce modules then we will take care of the authentication. "Authorization: Basic " + base64_encode(apikey)

Compliance

The Payment Card Industry Data Security Standard (PCI DSS) is a security standard for businesses that store, process or transmit credit cards. PCI DSS is a continuous process and since 2017, we have successfully completed two level 1 certifications and one SAQ D certification. You can view our current Attestation of Compliance (AOC) here.
